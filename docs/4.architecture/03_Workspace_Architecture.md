# Purpose

Workspace Framework 定義 Highlight AI 產品共同使用的 Workspace 架構。

Workspace 是所有使用者登入後工作的地方。

Workspace 不代表 Dashboard。

Workspace 是：

AI Decision Workspace。

所有產品皆應遵守相同 Workspace 設計原則。

---

# Philosophy

傳統 SaaS：

Login

↓

Dashboard

↓

Analysis

Highlight AI：

Login

↓

Workspace

↓

Decision

↓

Action

Workspace 的目的：

不是展示更多資料。

而是協助使用者完成工作。

# Workspace

Workspace 是企業的工作空間。

一個 Workspace

代表一個獨立的 Business Context。

Workspace 可包含：

- Organization
- Team
- Projects
- Data Sources
- AI Decisions
- Reports

所有資料皆屬於 Workspace。

---

# Workspace Home

Workspace Home 是登入後首頁。

Workspace Home

不是 Dashboard。

Workspace Home

是：

Decision Center。

Workspace Home 回答：

今天最重要的是什麼？

今天應先完成什麼？

目前企業是否健康？

Workspace Home 永遠以 AI 為中心。

---

# Workspace Home Structure

Workspace Home 建議包含：

Today's Signals

Top Recommendations

Evidence Preview

Recent Changes

Notifications

Workspace Home 不展示完整 Dashboard。

Dashboard 應延後至 Module。
